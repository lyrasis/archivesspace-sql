= Resource Queries
Useful queries for retrieving information about resource records.

Replace "https://archivesspace_base_url/" as appropriate to point to staff or public versions of records in link output columns.

== Queries
Retrieve all resource records' titles, ArchivesSpace IDs, **Identifier** values, and the count of how many archival objects they are linked to, sorted by most to least AOs:

[source,sql]
----
SELECT 
    r.title, CONCAT('http://archivesspace_base_url/', r.id) AS link, r.id, r.identifiers,
    COUNT(a.root_record_id) AS ao_count
FROM resource AS r
LEFT JOIN archival_object AS a
    ON r.id = a.root_record_id
GROUP BY r.id
ORDER BY ao_count DESC;
----

Retrieve the title, link, ArchivesSpace ID, and **Identifier** value of any resource record missing an abstract.

[source,sql]
----
SELECT 
    r.title,r.id,CONCAT('http://archivesspace_base_url/', r.id) AS link,
    r.identifier
FROM resource AS r
WHERE r.id NOT IN (
    SELECT r.id
    FROM resource AS r
    JOIN note AS n
        ON r.id = n.resource_id
    WHERE n.notes LIKE '%:"abstract"%'
);
----
